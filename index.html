Features

客户常用功能

NMEA 视图

显示原始 NMEA 数据和输出频率，便于检查模块实时输出。

信号视图

按卫星系统展示信号强度，辅助判断天线和测试环境。

消息视图

按 GGA、GLL、GSA、GSV、RMC、VTG 类型查看解析字段。

地图定位

根据有效定位数据显示红色定位箭头，支持拖动和缩放。

Requirements

运行要求

在线访问建议使用 Microsoft Edge 或 Google Chrome。串口连接依赖浏览器 Web Serial API，需要用户授权访问串口设备。

  • 推荐浏览器：Microsoft Edge / Google Chrome
  • 在线网站：适合客户快速打开和查看说明书
  • 本地服务：适合现场串口调试，运行打开GNSS网页版.bat
  • 说明书：CORVON GNSS Tool操作说明书v1.0.pdf
